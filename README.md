#=> Spring Autowiring Demo using @Autowired
This project demonstrates Autowiring in Spring using the @Autowired annotation.

## Description:
A Student object contains Certification details.
Spring automatically injects the Certification object into the Student object using @Autowired.

## Technologies:
* Java
* Spring Framework
* Spring Boot
* Maven

## Concepts Used:
* Spring IoC Container
* @Component
* @Autowired
* Component Scanning

## Output:
The Student bean is retrieved from the Spring container and prints student information along with certification details.
